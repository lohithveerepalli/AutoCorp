Getting started

1. Configure models

Open Settings / Models, pick providers per agent, review live costs, save.

2. Launch a company

Fill the CEO brief. Use demo auto-approve for first dry run without API keys.

3. Approve money & irreversible actions

Agents pause for domain purchases and live social creation unless auto-approved.

4. Iterate

Open Company detail → Run +2 cycles to keep the autonomous loops going.
